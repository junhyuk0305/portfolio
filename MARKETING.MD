## 🎯 3. 게시 운영 팁

| 항목 | 권장 사항 |
|---|---|
| **게시 시점** | D-10 (5/21), D-5 (5/26), D-2 (5/29) 3회 분산 |
| **에브리타임 게시판** | 자유게시판 · 정보게시판 · 비전공/취업 게시판 |
| **당근 게시판** | 동네생활(질문/일상) → 모임 → 과외/클래스 순으로 교차 |
| **카드뉴스 순서** | 1번(후킹) → 2번(솔루션) → 3번(CTA) 고정 — 순서 바꾸면 이탈률 ↑ |
| **댓글 운영** | "비전공인데 가능한가요?" 류 질문에 즉답 → 반응률 2~3배 ↑ |
| **A/B 테스트** | 제목만 다르게 2~3개 학교/동네에 분산 게시 → CTR 좋은 카피 본 캠페인에 채택 |
| **금지 표현** | "100% 취업 보장" "확정 합격" 등 과장 — 플랫폼 제재 + 신뢰 하락 |
```

PROLOGUE · 자기소개

병목을 끊어내는 사람

— 비즈니스와 개발 사이, 잃어버린 번역어를 찾아서.

비즈니스의 목적과 개발의 언어를 동시에 이해하며 프로덕트의 병목을 끊어내는 서비스 기획자 김태윤입니다. 경영학과 컴퓨터공학을 복수 전공하며 가설 수립부터 API 명세 작성, 스프린트 관리까지 풀사이클 기획 역량을 다졌습니다.

학내 동아리방 예약 서비스 'RoomSync' 기획 당시, 회장단 설문으로 '실시간 현황 확인'이라는 핵심 니즈를 데이터로 검증한 뒤 MVP 전략으로 기능을 과감히 추렸고, 출시 한 달 만에 30개 동아리 가입과 노쇼율 60% 감소를 만들었습니다. 또한 MAU 50만 커머스 스타트업 인턴십에서는 GA4와 사용성 테스트로 이탈 구간을 진단해 A/B 테스트를 기획했고, 장바구니 전환율을 12%p 끌어올린 B안이 실제 정규 배포로 이어지는 성과를 냈습니다.

이처럼 저는 데이터로 가설을 세우고 현장의 목소리로 검증하며, 개발팀과의 소통 비용을 직접 줄이는 기획자입니다.

데이터로 가설을 세우고, 현장의 목소리로 검증한다. — 김태윤, 기획 노트 중에서
FEATURE / 01 · RoomSync

RoomSync, 동아리방을 다시 설계하다

— 50개 동아리가 수기로 다투던 회의실, 한 달 만에 노쇼가 사라졌다.

2023.07 — 2023.11 · 30개 동아리 가입 · 노쇼 −60%

50여 개 동아리가 공용 회의실을 수기로 예약하다 겹침 사고가 빈번하게 일어났다. 회장단 설문을 돌려 '실시간 예약 현황 확인'이 최대 니즈임을 데이터로 확인했고, 백엔드 2명과 협업해 MVP 전략으로 핵심(예약·취소·실시간 뷰)만 남기고 알림은 2차 스프린트로 미뤘다.

API 명세는 직접 초안을 작성했고, Jira를 도입하며 스크럼 마스터를 맡았다. 출시 1개월 만에 30개 동아리가 가입했고 노쇼는 60% 줄었다. 이 글은 그 5개월의 기록이다.

FEATURE / 02 · A사 커머스 인턴십

12%p 전환율, 버튼 하나의 무게

— MAU 50만의 장바구니, 텍스트를 가리던 고정 버튼을 옮긴 결과.

2023.01 — 2023.06 · 장바구니 전환율 +12%p · 정규 배포

MAU 50만 커머스에서 장바구니 전환율 개선 과제를 맡았다. GA4와 앰플리튜드 분석으로 상세→장바구니 구간의 이탈이 45%에 달함을 확인했고, 사용자 5명 UT에서 하단 고정 구매 버튼이 본문 텍스트를 가린다는 페인포인트를 발견했다.

투명도와 높이를 조정한 안과 무료배송 넛지 카피를 더한 안을 A/B로 분기시켰다. B안이 12%p 상승했고, 그대로 정규 배포로 이어졌다.

FEATURE / 03 · 연합 창업 동아리

서로의 언어로 말하기

— 직무가 갈라진 100명의 동아리, 데모데이 배포율을 두 배 넘게 끌어올린 회장의 1년.

2022.03 — 2022.12 · 데모데이 배포율 40 → 85%

100명 규모 IT 연합 창업 동아리의 회장을 맡았을 때, 직무 간 교류 부재로 프로젝트 완성률은 40%에 머물러 있었다. '크로스오버 해커톤'과 '서로의 언어로 말하기' 세미나를 월 1회 운영하며 디자이너·개발자·기획자가 한 테이블에 앉도록 강제했다.

연말 데모데이에서는 85%의 팀이 실제로 제품을 배포했다. 사람 사이의 번역어를 만드는 것 또한 기획의 일이라는 사실을 그때 배웠다.
